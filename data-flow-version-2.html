1 — Vision
ESP32-CAMOV2640 sensor
Small camera module. Captures images and sends them over WiFi to the PC.
JPEG · WiFi
OpenCVHaar Cascade
Python library. Analyzes each image to find if a human face is present.

Lightweight AI model. Reads the face and predicts the emotion among 4 classes.

The detected emotion and confidence score are saved in RAM for later use.
2 — Audio
Microphone USBPCM 16kHz
USB mic plugged into the PC. Captures sound continuously at 16kHz.
audio stream
Porcupine"Desk Buddy"
Listens for the trigger word only. Uses less than 5% CPU. Works offline.
wake event
Vosk STToffline · ~1s
Converts your voice into text. Runs locally on the PC, no internet needed.
text + emotion
OllamaLlama 3.2 · local
Free local AI model. Receives your text + detected emotion and generates a smart response.
response ~2s
Intent + ResponseTASK / ANSWER / ...
The AI returns a text response and identifies what the user wants to do.
3 — Output
Response textfrom OpenAI
The text answer generated by the AI, ready to be spoken and displayed.
string
pyttsx3 TTSeSpeak · offline
Text-to-Speech library. Converts the text into voice audio. No internet needed.
PCM audio
Speakerjack 3.5mm
Small speaker connected to the PC audio jack. Plays the voice response out loud.
I2C · bitmap
OLED SSD1306128×64 px
Tiny screen connected via I2C. Displays the emotion face and the response text.
4 — Tasks & Reminders
Intent detectedTASK / REMINDER
The AI identified that the user wants to create a task or set a reminder.
INSERT
SQLitetasks.db
Local database stored on the PC. Saves tasks with title, date and time.
monitors
Schedulerthreading
Runs in background and checks if a task deadline has been reached.
triggered
AlertSpeaker + OLED
When the time comes, plays a voice alert and displays the reminder on screen.

OpenAI unreachableconnection error
The PC cannot reach the OpenAI server because there is no internet connection.
try / except
Local fallback"No connection"
Python catches the error. A default message is used instead of the AI response.
continues
